Submit cosmetic product notifications by uploading CPNP ZIP files

The simplest way to provide cosmetic product information is to upload ZIP files exported from CPNP.

How to upload a ZIP file from CPNP

  1. export your ZIP files from CPNP, read how to export a CPNP ZIP file
  2. upload as many ZIP files at once as you like
  3. add formulation information for any files that had formulation information attached as a separate document on CPNP
  4. add images of the packaging
  5. confirm responsibility

The process is the same for claiming responsibility for products on sale before Brexit and new products.
